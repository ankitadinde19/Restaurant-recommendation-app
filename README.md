# 🍽️ Restaurant Recommendation App

# Project Description
This is a Flask-based web application that recommends restaurants based on:
- Restaurant name
- Cuisine
- Budget
- Rating

# Features
- Content-based recommendation system (TF-IDF + Cosine Similarity)
- Filter by budget and rating
- Clean UI using Flask templates
- Analytics graph

# Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS

# How to Run
```bash
pip install -r requirements.txt
python app.py
